## 有多少数字货币丢失了

因忘记秘钥、助记词或者丢失存储秘钥的硬件，而丢失比特币的事件数不胜数。根据数字取证公司对比特币区块链进行的研究，大约有288万至399万个比特币已经永远消失，对应比特币总数的17%至23%, 到2040年左右，比特币的总挖掘量将达到2000万个左右，但实际上可流通的比特币将远低于这一数字。
